# Strawberry_Cafe
A responsive, static website for a modern cafe. Features include:  Custom #ff43a4 color theme.  Product menu with category filtering (Coffee, Cake, Sodas).  Functional "Add to Cart" sidebar (vanilla JS).  Optimized CSS (Flexbox/Grid) for consistent product card alignment. Ready for Cloud Deployment.  Tech Stack: HTML5, CSS3, Vanilla JavaScript
